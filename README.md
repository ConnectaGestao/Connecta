# Gestão Central

Sistema completo de Gestão Central, incluindo:
- Cadastro e acompanhamento de cidadãos (Munícipes).
- Lançamento de Atendimentos com regras de valores e datas.
- Controle e acompanhamento de Campanhas (WhatsApp).
- Painel de Administração para personalizar Especialidades, Procedimentos (com valores automáticos), e outros.
- Exportação de dados e Relatórios detalhados.
- Controle de acesso unificado via Firebase Auth e Firestore.

## Tecnologias
- **Frontend:** HTML5, TailwindCSS (via CDN), JavaScript Puro (Vanilla).
- **Componentes Visuais:** Lucide Icons, Chart.js, Flatpickr, TomSelect.
- **Backend/DB:** Firebase (Auth, Firestore).
- **Hospedagem Recomendada:** GitHub Pages.

## Instalação
Basta hospedar os arquivos estáticos (HTML, JS, CSS) em qualquer servidor web ou plataforma como GitHub Pages/Vercel.
Nenhuma compilação (build step) é necessária.
